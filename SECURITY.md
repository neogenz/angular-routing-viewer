# Security Policy

## Scope

ngrv is a developer CLI that performs **static analysis** of Angular source files and emits a self-contained HTML file. It does not execute the analyzed project's code, run a server, or make network requests.

## Supported versions

Only the latest published version on npm receives fixes.

## Reporting a vulnerability

Please do not open a public issue for security problems.

Report privately via [GitHub Security Advisories](https://github.com/neogenz/angular-routing-viewer/security/advisories/new), or email **maxime.desogus@gmail.com**.

Include a description, reproduction steps, and the impact you observed. You can expect an initial response within a few days. Once a fix is released, credit will be given in the changelog unless you prefer to remain anonymous.
